# AKRS Memory — Index

> Memory files are **indexes**, not documentation. Each points to the one owner of a
> piece of knowledge. No duplication. Spec: `docs/work-flow-ai/Constitution.md`.

| Memory | Owns the pointer to… |
|---|---|
| `identity.md` | identity & contact facts + their code owner |
| `content-layer.md` | the typed content modules under `app/content/` |
| `projects.md` | featured/secondary set + the two-register AI rule |
| `design-system.md` | visual & motion direction |

**Authoritative knowledge always lives outside Memory:**
- Facts → `docs/source-of-truth.md`
- Build spec → `docs/requirements-and-design.md`
- Data → `app/content/*`
